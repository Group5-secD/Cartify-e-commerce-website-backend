# Cartify E-commerce Website - Backend

**Cartify Backend** provides the server-side functionality for the Cartify e-commerce platform using **PHP**. It handles product management, user authentication, cart operations, and order processing while supporting the frontend shopping experience.

The backend focuses on implementing core server-side concepts such as API design, database interaction, and secure authentication without relying on heavy frameworks.

---

## Project Overview

This backend service allows the application to:

- Retrieve and manage product data
- Handle user authentication
- Manage shopping cart operations
- Process orders
- Provide API endpoints for the frontend

It aims to demonstrate understanding of backend development using PHP and database integration.

---

## Features

- REST-style API endpoints
- User authentication and session management
- Product and cart management
- Order processing
- Database integration for persistent storage

---

## Technologies Used

- **PHP**
- **MySQL**

---

## API Integration

The backend communicates with the Cartify frontend through HTTP requests. It sends and receives data in **JSON format** to handle product information, user authentication, cart updates, and order processing.
